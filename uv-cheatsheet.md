# UV Cheat Sheet

This cheat sheet provides a quick reference to commonly used commands and features in UV, a text editor designed for programmers.

[Astral: UV docs](https://docs.astral.sh/uv/)

| Command | Description |
|---------|-------------|
| `uv python list` | Lists all available Python versions installed on the system. |
| `uv run (--python <version>) <script.py>` | Executes the specified Python script using UV's built-in interpreter. |
| `uv add <package>` | Installs a Python package using UV's package manager. |
| `uv remove <package>` | Uninstalls a Python package using UV's package manager. |
| `uv update <package>` | Updates a specified Python package to the latest version. |
| `uv search <package>` | Searches for a Python package in the UV package repository. |
| `uv init` | Initializes a new UV project in the current directory. |
| `uv sync` | Synchronizes the project dependencies with the `uv.lock` file. |
